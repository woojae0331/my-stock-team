# my-stock-team — AI 애널리스트 팀이 만들어주는 주식 리서치 📈

**종목 이름만 입력하면, 여러 명의 AI 애널리스트가 역할을 나눠 분석하고,
그 결과를 보고서와 발표용 슬라이드(PPT)로 만들어주는 도구입니다.**

> 💡 이 도구는 **공부·연습용**입니다. 무료 공개 데이터를 사용하며, **투자를 권유하지 않습니다.**
> "이 주식을 사라/팔라"는 말은 절대 하지 않고, **판단에 필요한 자료만 정리**해 드립니다. 최종 결정은 사람이 합니다.

---

## 이게 뭔가요? (1분 설명)

한 명의 사람이 주식 보고서를 쓰려면 재무제표도 보고, 주가 차트도 보고, 뉴스도 찾고, 위험요소도 따져봐야 합니다.
이 도구는 그 일을 **5명의 AI 전문가**가 나눠서 합니다.

| AI 전문가 | 하는 일 |
|-----------|---------|
| 📊 펀더멘털 애널리스트 | 회사의 재무(매출·이익)와 가치를 분석 |
| 📈 차트 애널리스트 | 주가 흐름과 추세를 분석 |
| 📰 뉴스·심리 애널리스트 | 최근 뉴스와 시장 분위기를 정리 |
| ⚠️ 리스크 애널리스트 | 위험요소 3가지를 짚어줌 |
| ✅ 검증 애널리스트 | 보고서에 틀린 곳·빠진 곳이 없는지 최종 점검 |

마지막으로 결과를 모아 **보고서(문서)** 와 **발표 슬라이드(PPT 파일)** 로 만들어 줍니다.

---

## 시작하기 전에 준비물 3가지

아래 3가지만 준비하면 됩니다. 천천히 따라 하세요. (각 단계는 한 번만 하면 됩니다.)

### ① Claude Code 설치하기
이 도구는 **Claude Code**라는 프로그램 위에서 돌아갑니다. 먼저 이걸 설치하세요.
- 설치 안내: <https://docs.claude.com/claude-code>
- 설치하면 컴퓨터에서 Claude와 대화하며 명령을 내릴 수 있게 됩니다.

### ② 무료 데이터 열쇠(키) 발급받기
회사 재무 데이터는 금융감독원의 **DART**라는 공식 사이트에서 가져옵니다.
이 데이터를 쓰려면 **무료 인증키**(긴 영어+숫자 문자열)가 필요합니다. 이메일만 있으면 5분이면 됩니다.

1. <https://opendart.fss.or.kr> 접속
2. 상단 **[인증키 신청/관리] → [인증키 신청]** 클릭
3. 이메일 등 간단한 정보 입력 → 신청
4. 메일로 받은 **인증키**(예: `a1b2c3...` 같은 40자리 문자열)를 복사해 둡니다. → ④번에서 사용

> 🔑 이 키는 **비밀번호 같은 것**입니다. 남에게 보여주거나 인터넷에 올리지 마세요.

### ③ 파이썬(Python) 설치하기 *(발표 슬라이드를 만들 때만 필요)*
PPT 파일을 만들려면 **파이썬**이라는 프로그램과 작은 부품 하나가 필요합니다.
- 파이썬 설치: <https://www.python.org/downloads/> 에서 다운로드 후 설치
  (설치 화면에 **"Add Python to PATH"** 체크박스가 있으면 꼭 체크하세요.)
- 설치 후, Claude Code 입력창에 아래를 입력하면 부품이 깔립니다:
  ```
  ! pip install python-pptx
  ```
  (맨 앞의 `!` 는 "이 명령을 컴퓨터에서 실행해줘"라는 뜻입니다.)

## 도구 설치하기 (2줄)

Claude Code 입력창에 아래 두 줄을 차례로 입력하세요.

```
/plugin marketplace add woojae0331/my-stock-team
```
```
/plugin install my-stock-team@my-stock-team
```

- 첫 줄: 이 도구가 있는 "앱 가게(마켓플레이스)"를 등록합니다.
- 둘째 줄: 거기서 `my-stock-team`을 설치합니다.

---

## 데이터 열쇠(키) 넣어두기

②에서 받은 키를 컴퓨터가 읽을 수 있게 **`.env`라는 메모장 파일**에 적어둡니다.
(`.env`는 "비밀 설정을 적어두는 파일"이라고 생각하면 됩니다.)

- 작업하는 폴더 안에 `.env` 라는 이름의 파일을 만들고, 아래 한 줄을 적은 뒤 저장하세요.
  (`발급받은_키` 자리에 ②에서 복사한 실제 키를 붙여넣으세요.)

```
DART_KEY=발급받은_키
```

> ⚠️ 이 파일은 **절대 GitHub 등에 올리면 안 됩니다.** (이 도구는 실수로 올라가지 않도록 미리 막아두었습니다.)
> 그냥 잘 모르겠으면, Claude Code에게 **"내 DART 키를 .env에 넣어줘"** 라고 말하면 도와줍니다.

---

## 사용하기 (가장 쉬운 부분 🙂)

Claude Code 입력창에 종목 이름과 함께 명령을 입력하면 끝입니다.

**1) 보고서 만들기**
```
/my-stock-team:analyze 삼성전자
```
→ AI 5명이 분석해서 `reports/삼성전자.md` 라는 보고서 문서를 만들어 줍니다.

**2) 발표 슬라이드(PPT)로 내보내기**
```
/my-stock-team:export-pptx 삼성전자
```
→ 위 보고서를 예쁘게 디자인한 `reports/삼성전자.pptx` 슬라이드로 만들어 줍니다.

> 종목 이름 자리에 원하는 한국 주식(예: `SK하이닉스`, `현대차`)을 넣으면 됩니다.

### 결과물은 어디에 있나요?
작업 폴더 안의 **`reports` 폴더**에 생깁니다.
- `종목명.md` → 글로 된 보고서 (메모장·VS Code 등으로 열기)
- `종목명.pptx` → 파워포인트 슬라이드 (PowerPoint·한쇼 등으로 열기)

## 꼭 알아두기 (중요)

- 📚 **공부·참고용입니다.** 실제 투자 권유가 아닙니다. 모든 결정과 책임은 본인에게 있습니다.
- 🔎 **출처 없는 숫자는 쓰지 않습니다.** 모든 수치 옆에 어디서 왔는지(출처·날짜)를 표시합니다.
- 🤷 데이터를 못 구하면 지어내지 않고 **"확인 불가"** 라고 솔직히 적습니다.
- 🚫 "사세요/파세요/목표가 ○○원" 같은 **단정적인 말은 하지 않습니다.** 판단 재료만 정리해 드립니다.

## 잘 안 될 때 (자주 묻는 것)

| 증상 | 이렇게 해보세요 |
|------|----------------|
| `/my-stock-team:...` 명령이 안 보여요 | 위 "도구 설치하기" 2줄을 다시 실행했는지 확인하세요. |
| 재무 데이터를 못 가져와요 | ④의 `.env` 파일에 `DART_KEY`가 제대로 들어갔는지 확인하세요. |
| PPT 만들 때 오류가 나요 | ③의 파이썬과 `pip install python-pptx`가 설치됐는지 확인하세요. |
| 그래도 모르겠어요 | Claude Code에게 **그냥 평소 말로** "○○ 때문에 안 돼"라고 물어보세요. 같이 고쳐줍니다. |

## 용어 한 줄 사전

- **Claude Code**: AI(클로드)와 대화하며 컴퓨터 작업을 시키는 프로그램.
- **플러그인**: Claude Code에 기능을 추가하는 "확장팩". 이 도구가 바로 플러그인입니다.
- **DART**: 금융감독원이 운영하는 기업 공시·재무 공식 사이트.
- **.env 파일**: 비밀 키 같은 설정을 적어두는 파일.
- **`.md` / `.pptx`**: 각각 글 보고서 파일 / 파워포인트 슬라이드 파일.
